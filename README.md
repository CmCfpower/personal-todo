# To-do

Локальная веб-страница для ежедневных списков задач.

## Как пользоваться

Открой `index.html` в браузере или запусти локально:

```powershell
node server.js
```

После запуска страница будет доступна по адресу `http://127.0.0.1:8765/`.

На странице можно:

- переключаться между списками по датам;
- открыть или создать список через поле даты;
- ставить галочки мышкой;
- отмечать статус задачи;
- оставлять комментарии к задачам;
- назначать дату задачи;
- автоматически переносить задачи со статусом `В процессе` на следующий день;
- переносить данные между доменами через экспорт и импорт JSON;
- фильтровать важные, открытые и выполненные задачи.

Данные сохраняются в браузере автоматически через `localStorage`.

## Файлы

- `index.html` — страница приложения.
- `styles.css` — внешний вид.
- `app.js` — логика и сохранение.
- `server.js` — простой локальный сервер для запуска через `localhost`.
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages.

Markdown-архивы вида `todo-YYYY-MM-DD.md` хранятся только локально и не публикуются в GitHub.

## GitHub Pages

В настройках репозитория открой `Settings` -> `Pages`.

Выбери:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

После сохранения сайт будет доступен по адресу `https://cmcfpower.github.io/personal-todo/`.
